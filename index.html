FILE COMPRESSOR

A project that performs Lossless Data Compression.

Compressed data can be decoded ensuring no ambiguity in the decoding.

This project is to design compression and decompression programs based on Huffman Coding. The idea of Huffman Coding is to minimize the weighted expected length of the code by means of assigning shorter codes to frequently-used characters and longer codes to seldom-used code. Built using HTML,CSS and JavaScript.
